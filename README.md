Bank Management System (Java + Swing + MySQL)

• Desktop-based ATM system built using Java Swing and MySQL  
• Secure login with auto-generated card number and PIN  
• Supports deposit, withdrawal, fast cash, and balance enquiry  
• Includes PIN change with proper validation  
• Stores data using MySQL through JDBC  
• Implements OOP concepts, CRUD, and exception handling  
• Ensures reliable and error-free transaction processing  
• Configure MySQL and update JDBC connection file  
• Run the project by executing Login.java from any IDE
